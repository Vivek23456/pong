f# Pong

A simple Pong game written in Rust.

## Description

This project is a Pong game built using Rust. The game implements the classic table-tennis style arcade game where two players control paddles and try to bounce the ball past their opponent.

## Features

- Two player gameplay
- Simple controls
- Score tracking
- Written in Rust

## Controls
